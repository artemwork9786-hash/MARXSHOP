# Фикс белых полос по краям карточки

## Причина
`backdrop-filter` + `overflow: hidden` + `border-radius` на корневом div → sub-pixel артефакт на стыке blur-слоя и края карточки.

## Решение — убрать `overflow: hidden` с корня, перенести на секции

**Файл:** `src/components/AccountCard.jsx`

### 1. Корень карточки (строка ~508)
Убрать `overflow-hidden`:
```
Было: ...relative overflow-hidden rounded-2xl border...
Стало: ...relative rounded-2xl border...
```

### 2. Видео секция — добавить `overflow-hidden`
Video player (GlassPlayer) уже имеет `overflow-hidden` на containerRef. Для случая без видео (строка ~513):
```
<div className="relative w-full aspect-video bg-[#111] overflow-hidden">
```

### 3. Блюр-фон цены — добавить `overflow-hidden`
Строка ~623:
```
<div className="relative bg-[#1A1A1A] overflow-hidden">
```

### 4. Extra Info оверлей — вернуть z-50 (было z-[60] в незакоммиченном изменении)
Оверлей уже имеет `rounded-2xl overflow-hidden` — артефакта не будет.

## Верификация
1. `npm run build && npm run deploy`
2. Открыть/закрыть "Доп. инфо" — белых полос по краям быть не должно
3. Проверить что видео не вылезает за границы карточки
