# Фикс: вернуть blur на нижнюю панель карточки

## Проблема
Нижняя панель карточки (цена + кнопка) потеряла blur-эффект. В Telegram WebView backdrop-blur не работает, поэтому нужна альтернатива.

## Решение

**Файл:** `src/components/AccountCard.jsx`

Нижняя панель (строка 667) уже имеет размытое фоновое изображение (`poster blur-xl opacity-30`), но сам контейнер не имеет `backdrop-blur-md`. В WebView это не работает.

**Фикс:** Добавить `backdrop-blur-md` и полупрозрачный фон на контейнер нижней панели:

```jsx
// Строка 667 — текущий код:
<div ref={bottomBarRef} className={`relative z-50 flex gap-2 px-3 sm:px-4 py-3 ${isCompact ? "flex-col" : "flex-row items-center justify-between"}`}>

// Заменить на:
<div ref={bottomBarRef} className={`relative z-50 flex gap-2 px-3 sm:px-4 py-3 backdrop-blur-md bg-black/40 ${isCompact ? "flex-col" : "flex-row items-center justify-between"}`}>
```

## Верификация
1. Карточка с видео: нижняя панель имеет blur-эффект поверх видео
2. `npm run deploy`
