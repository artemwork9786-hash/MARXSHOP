# Plan: Resume MiMoCode Session by ID

## Goal
The user wants to resume a specific MiMoCode session with ID `ses_080dfea7effetE6PuDSxz0GaLW`.

## Finding
No code changes needed — this is a CLI usage question. The answer is documented in the mimocode-docs skill.

## Solution
Run the following command from the terminal:

```bash
mimo --session ses_080dfea7effetE6PuDSxz0GaLW
```

or shorthand:

```bash
mimo -s ses_080dfea7effetE6PuDSxz0GaLW
```

This launches the TUI and restores the conversation context (memory, checkpoints, task tree) for that session.
