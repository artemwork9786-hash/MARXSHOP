# План: Исправление плашки звука на мобильных

## Проблема
На мобильных устройствах плашка громкости (volume slider):
1. Не открывается — кнопка громкости имеет только `onMouseEnter/Leave`, нет `onClick`
2. Не скрывается вместе с панелью контролов — `showVolume` не сбрасывается при `showControls = false`
3. Взаимодействие с ползунком не продлевает жизнь контролов

## Файл для изменения
`src/components/AccountCard.jsx` — компонент GlassPlayer

## Изменения

### 1. Добавить `onClick` на кнопку громкости (строка ~380)
Текущий код:
```jsx
<button ref={volumeBtnRef} onClick={toggleMute} onMouseEnter={...} onMouseLeave={...}>
```
Нужно: при клике на телефоне — toggle `showVolume`, на десктопе — toggleMute (или отдельная логика).

Решение: `onClick` делает toggle `showVolume` (так как на десктопе уже есть hover). Плюс вызывает `resetHideTimer()`.

### 2. Сбрасывать `showVolume` при скрытии контролов (строка ~272)
В `resetHideTimer` добавить `setShowVolume(false)`.

### 3. Сбрасывать `hideTimer` при взаимодействии с плашкой (строка ~394)
На `onPointerDown` плашки звука — вызывать `resetHideTimer()`.

### 4. Сбрасывать `hideTimer` при клике на кнопку громкости
В `onClick` кнопки громкости — вызывать `resetHideTimer()`.

## Верификация
1. На телефоне: тап по иконке громкости → плашка появляется
2. На телефоне: тап на плашке → ползунок работает, контроли остаются
3. На телефоне: 1с бездействия → контроли + плашка исчезают одновременно
4. На десктопе: hover по иконке громкости → плашка появляется (как раньше)
5. На десктопе: увод мышки → плашка исчезает через 300мс
